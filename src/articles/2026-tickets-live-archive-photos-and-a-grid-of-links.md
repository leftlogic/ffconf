## Latest Event News

We have **[launched tickets](https://2026.ffconf.org)** with a reduced early bird price of £219+VAT! There's a limited number available so grab yours and please spread the word 💜

**[So join us on Friday 13th November, and become a better citizen on the web.](https://2026.ffconf.org)**

## Community News

This week's community news comes from the corners of CSS grid. There's a couple of interesting specifications landing in browsers and there's some equally interesting discussions.

- [CSS grid lanes](https://gridlanes.webkit.org/) - aka masonry, landed in Safari and this is a comprehensive set of demos
- Jake Archibald [raises concerns](https://bsky.app/profile/jakearchibald.com/post/3mjw3drcf4s2p) about the use case for CSS grid lanes
- Manuel Matuzovič calls out the [accessibility issues](https://matuzo.at/blog/2026/grid-lanes-accessibility) inherent in CSS grid lanes
- Remy confused CSS grid lanes for [CSS gap decorations](https://utilitybend.com/blog/css-is-filling-the-gaps-with-rules-a-way-to-style-gaps-in-grid-and-flex)
