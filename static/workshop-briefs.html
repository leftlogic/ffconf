Universal React using Next.js with me, Remy Sharp

What's "universal React?" - it's React JavaScript that you only have to write once, and it runs both in the browser, but also the server. This offers huge benefits like SEO, improved performance to the visitor, and reduced code.

Using Next.js, universal React requires very little overhead to create. The workshop includes: instant server side rendered apps, connecting to databases, advanced routing, testing with jest (and gotchas) and deploying strategies.
