Universal React using Next.js with me, Remy Sharp

The promise of Universal React? The same code running in the browser and the client? Next.js makes it easy to write JavaScript that you only have to write once, and it runs both in the browser and on the server. This offers huge benefits like SEO, improved performance to the visitor, and reduced code.

Using Next.js, universal React requires very little overhead to create. The workshop includes: instant server side rendered apps, connecting to databases, advanced routing, testing with jest (and gotchas) and deploying strategies.
